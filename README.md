# Sistema de Gestión de Incidencias y Sanciones

Aplicación web desarrollada para la gestión, seguimiento y resolución de faltas al reglamento interno. Este sistema permite a personal autorizado registrar infracciones y asignar horas de servicio comunitario a los usuarios correspondientes, manteniendo un control estricto del cumplimiento.

## Características Principales

* **Arquitectura de Roles:** Control de acceso y permisos diferenciados para Alumnos, Profesores y Personal de Seguridad.
* **Gestión de Infracciones:** Módulo para levantar reportes de faltas en tiempo real, asociándolas directamente al perfil del infractor.
* **Seguimiento de Sanciones:** Asignación y control de horas de servicio comunitario como método de resolución de infracciones.
* **Sistema de Notificaciones:** Alertas globales integradas para mantener informados a los usuarios sobre nuevos lineamientos o avisos importantes.
* **Módulo de Reportes:** Generación de resúmenes de incidencias para evaluación administrativa.

## Stack Tecnológico

* **Backend:** PHP (Vanilla/Nativo) - *Demuestra dominio de las bases del lenguaje, manejo de sesiones y consultas directas.*
* **Base de Datos:** MySQL - *Diseño relacional para conectar usuarios, catálogos de faltas y registros de servicio.*
* **Frontend:** HTML5, CSS3, JavaScript.

## Propósito del Proyecto
Este sistema fue desarrollado para solucionar un problema real de logística administrativa, automatizando un proceso manual y garantizando la trazabilidad de las sanciones desde que se comete la falta hasta que se cumplen las horas de servicio.
